# MC Language Datagen

Datagen for Minecraft Languag JSON in JS

[Back to the Homepage](../..)


### Example Usage

```sh
mc-lang-dt --project my_mod --lang de_de \
  --entry hello Hallo \
  --entry bye Tschüss
```

### Install

```sh
git clone https://github.com/ShadowDara/Awesome-Minecraft-Tools

cd content/language_datagen

pip install -e .
```
